CSS Box Model Reference

Box Model

Everything in CSS has a box around it, and understanding these boxes is key to being able to create layouts with CSS, or to align items with other items

Box Model Properties

Part of Box Properties Description/Details
Content Box width, height The area where your content is displayed, sized using these properties
Padding Box padding The padding sits around the content as white space; its size can be controlled by this property
Border Box border The border box wraps the content and any padding. Its size and style can be controlled using this property
Margin Box margin The margin is the outermost layer, wrapping the content, padding, and border as whitespace between this box and other elements. Its size can be controlled using this property
